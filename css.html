CSS

CSS stands for Cascading Style Sheet

Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML.

Format

There are three types: Inline, Internal, External

Inline Styling: using the style attribute inside HTML elements. For example: < h1 style= "color: black;" >


Internal - by using a < style >< /style > element in the < head > section.

CSS Properties

Property Use
text-decoration: Used to put lines under, over or striking the text,overline is used to put a line over the text, underline is used to put a line under the text. line-through is used to strike the text.
font-family: It is used to change the font of the text, for example, Ariel.
font-size: It is used to change to size of the text, while writing the value use px after the number.
color: It is used to change the color of the text.
background-color: It is used to change the background colour of the element
font-weight: It sets the boldness of the text.
text-align: It specifies the horizontal alignment of text in an element.
font-style: It specifies the font style for a text. For exapmle, italic.
