HTML

HTML stands for Hyper-Text Markup Language

It was made by Tim Berners Lee in 1993.
HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

Tags Format

< !doctype html > refers to the latest version of HTML.
There are 2 types of tags in HTML: Empty tages and Container tags. Container tags act as a block of text, it holds value in it, for example: < p >< /p > and < h1 >< /h1 >. They have an opening and closing tag. Empty tags don't have a closing tag, for example: < img > used for adding images in the webpage.
< html >< /html > is the beginning of an HTML document. It is the starting code block of websites. It consists of 2 parts: head and body.
< head > < /head > contains machine-readable information about the HTML document, like its title, scripts, and style sheets. It is not shown in the website you have created.
< body >< /body > contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. It is shown in the website you have created.
Heading Tags have 6 sizes: < h1 >< /h1 >, < h2 >< /h2 >, < h3 >< /h3 >, < h4 >< /h4 >, < h5 >< /h5 >, < h6 >< /h6 >. These are used for headings in the body tag.
< p >< /p > is used to type paragraphs into the webpage.
